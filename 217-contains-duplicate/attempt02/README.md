# LeetCode 217 - Contains Duplicate

This repository contains my second attempt at LeetCode problem 217 **Contains Duplicate**.

The goal of the problem is to return a boolean which indictes if nums contains a duplicate value.

## Current approach

Uses the std::sort function to sort the array then check whether the item is next to a duplicate.
I decided to use C++ instead of Java as I felt the sorting function would be faster for this approach.
