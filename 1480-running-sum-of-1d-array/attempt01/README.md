# LeetCode 1480 - Running Sum of 1D Array

This repository contains my first working attempt at LeetCode problem 1480, "Running Sum of 1D Array".

The goal of the problem is to return a new array where each value is the sum of the values from the start of the original array up to the current index.

For example:

```text
Input:  [1, 2, 3, 4]
Output: [1, 3, 6, 10]
```

## Project Structure

```text
src/
  Main.java
```

`Solution.java` contains:

- a `main` method with a sample input array
- a `runningSum` method that calculates the running sum
- console output for each value in the result

## Notes

This is intended to stay as a first attempt rather than a polished final solution. The code works for the sample input and shows the basic idea clearly.
