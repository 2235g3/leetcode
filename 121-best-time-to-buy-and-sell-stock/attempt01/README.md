# LeetCode 121 - Best Time to Buy and Sell Stock

This repository contains my first attempt at LeetCode problem 121 **Best Time to Buy and Sell Stock**.

The goal of the problem is to return the maximum profit achievable by choosing a single day to buy and a future day to sell.

## Current approach

Store the minimum price found each and then determine if the profit to be made is more than the current maximum.
