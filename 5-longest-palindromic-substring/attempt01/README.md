# LeetCode 5 - Longest Palindromic Substring

This repository contains my first attempt at LeetCode problem 5, **Longest Palindromic Substring**.

The aim of this repo is to keep a record of the attempt as it was written, rather than to present a polished or optimised final solution.

## Problem

Given a string, return the longest palindromic substring contained within it.

A palindrome is a sequence of characters that reads the same forwards and backwards.

## Notes

- This is a failed attempt and is intentionally left as a snapshot of the learning process.
- The current approach does not reliably return a valid palindrome for all inputs.
- There are no automated tests in this repository.
- The project was created as a simple IntelliJ Java project.
